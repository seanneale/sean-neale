Premier Manager 2

Premier Manager 2 is a web-based football management game using real players and abilities based on real stats. The aim is to pick to a team based on your squad and go on to win the league. At the end of the first two weeks of the WDI course I built a much smaller version of this game entirely browser based. When the end of the course came round I decided I wanted to have improve on this original project. My aims for the second attempt were to:

  • The ability to save games
  • To use real data to generate player info
  • To allow the user to take part in multiple leagues
  • To allow multiple users to take part in each game (Unfortunately I had to temporarily abandon this one in favour of finishing the single user experience
I used the following to build the game
  • HTML,CSS
  • JavaScript
  • Ruby On Rails
  • Angular JS
  • Sidekiq/Redis - Background jobs
  • Official Fantasy Premier League Data
  • Championship Manager/Football Manager for inspiration!
